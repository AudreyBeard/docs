# wandb artifact

**사용법**

`wandb artifact [옵션] 코맨드 [ARG]...`

**요약**

Artifacts와 상호작용하는 명령어

**옵션**

| **옵션** | **설명** |
| :--- | :--- |

**코맨드**

| **코맨드** | **설명** |
| :--- | :--- |
| cache | 아티팩트 캐시와 상호작용하는 명령어 |
| get | wandb에서 아티팩트 다운로드 |
| ls | wandb 프로젝트의 모든 아티팩트 나열 |
| put | wandb에 아티팩트 업로드 |
