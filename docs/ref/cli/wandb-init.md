
# wandb init

**使用方法**

`wandb init [OPTIONS]`

**概要**

Weights & Biases でディレクトリーを設定

**オプション**

| **オプション** | **説明** |
| :--- | :--- |
| -p, --project | 使用するプロジェクト。 |
| -e, --entity | プロジェクトをスコープするEntity。 |
| --reset | 設定をリセット |
| -m, --mode | "online"、"offline" または "disabled" のどれか。デフォルトは online。 |
