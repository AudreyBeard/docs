| Markdown       | Shortcuts     |  
|----------------|----------------|  
| Headers        | `#`, `##`, `###`, etc.  |  
| Bold           | `**text**` or `__text__` |  
| Italics        | `*text*` or `_text_`     |  
| Strikethrough  | `~~text~~`     |  
| Unordered List | `* item` or `- item`   |  
| Ordered List   | `1. item`      |  
| Code           | `` `code` ``    |  
| Blockquote     | `> quote`      |  
| Link           | `[text](URL)`  |  
| Image          | `![alt text](URL)` |
