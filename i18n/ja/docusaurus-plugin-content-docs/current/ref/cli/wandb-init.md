# wandb init

**使い方**

`wandb init [OPTIONS]`

**概要**

Weights & Biasesでディレクトリを設定する

**オプション**

| **オプション** | **説明** |
| :--- | :--- |
| -p, --project | 使用するプロジェクト。 |
| -e, --entity | プロジェクトを適用するエンティティ。 |
| --reset | 設定をリセットする |
| -m, --mode | "online"、"offline"、または "disabled" にすることができます。デフォルトは online。 |
| --help | このメッセージを表示し、終了します。 |
