# wandb login

**使い方**

`wandb login [オプション] [キー]...`

**概要**

Weights & Biasesにログインします。

**オプション**

| **オプション** | **説明** |

| :--- | :--- |

| --cloud | ローカルではなくクラウドにログインする |

| --host | W&Bの特定のインスタンスにログインする |

| --relogin | すでにログインしている場合、再ログインを強制する |

| --anonymously | 匿名でログインする |

| --help | このメッセージを表示し、終了する |
